SUSE
****

The `openSUSE project <http://en.opensuse.org/Project_Overview>`_ is a worldwide community program sponsored by `Novell <http://www.novell.com/>`_ that promotes the use of Linux everywhere. The program provides free and easy access to openSUSE. Here you can find and join a community of users and developers, who all have the same goal in mind â€” to create and distribute the world's most usable Linux. openSUSE also provides the base for Novell's award-winning `SUSE Linux Enterprise <http://www.novell.com/linux/>`_ products.
